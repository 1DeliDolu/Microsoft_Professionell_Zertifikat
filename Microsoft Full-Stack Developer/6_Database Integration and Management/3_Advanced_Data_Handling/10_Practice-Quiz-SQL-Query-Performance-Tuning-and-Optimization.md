### 2. Question 2

Which of the following techniques helps limit the amount of data processed by SQL queries?

* [ ] Aggregating data with functions like COUNT or AVG
* [ ] Using SELECT * to retrieve all columns
* [ ] Using joins to merge tables
* [X] Using a WHERE clause to filter results

That’s correct! A WHERE clause limits results based on specified criteria, reducing the data the database processes.

### 3. Question 3

Which SQL optimization technique involves creating a data structure to improve query performance by organizing data on specific columns?

* [ ] Using foreign keys to link tables
* [ ] Normalization
* [ ] Database partitioning
* [X] Indexing

That’s correct! Indexing creates a structured data arrangement on specified columns, allowing the database to locate rows and improve performance quickly.
