
### 1. Question 1

A company stores sensitive customer information, including payment details, in its SQL database. What is the primary reason for implementing strong database security measures?

* [ ] To allow unrestricted access for all employees
* [ ] To improve database visualizations
* [X] To protect sensitive data from unauthorized access

That’s correct! Protecting sensitive data is crucial to prevent data breaches, financial loss, and reputational damage.

* [ ] To ensure faster query performance

### 2. Question 2

An online retail application allows customers to search for products using text inputs. To prevent SQL injection attacks, which approach should the development team prioritize?

* [ ] Data caching
* [ ] Replicating the database
* [ ] Index optimization
* [X] Parameterized queries

That’s correct! Parameterized queries prevent SQL injection by treating user input as data, not executable code.
